# Sample CVs Folder

This folder contains sample CV files in various formats (PDF, Word, Text) that will be used by the Fair-Hire Sentinel system for bias detection and semantic analysis.

## Supported Formats
- PDF (.pdf)
- Microsoft Word (.docx, .doc)
- Text files (.txt)

## Usage
The system will automatically scan this folder and process all CV files for:
- ATS screening simulation
- Bias detection analysis
- Semantic matching
- Candidate rescue alerts

Add your CV files here and they will be included in the analysis.
