# Fair-Hire Sentinel - Frontend

Next.js frontend for the Fair-Hire Sentinel AI-powered bias detection dashboard.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Pages

- **Home** (`/`) - Overview dashboard with key metrics and alerts
- **Dashboard** (`/dashboard`) - Real-time monitoring view
- **Analytics** (`/analytics`) - Deep dive into bias patterns
- **About** (`/about`) - Information about the platform
- **Contact** (`/contact`) - Contact form and information

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Recharts (for visualizations)
